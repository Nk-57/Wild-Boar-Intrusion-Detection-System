# Predictive Intrusion Detection and Collective Deterrence for Wild Boar Prevention

## Overview

This project is a low-cost solar-powered embedded system designed to detect wild boar intrusion in agricultural fields and activate deterrent mechanisms.

## Problem Statement

Wild boar intrusion causes severe crop damage in rural agricultural areas. Farmers require affordable and reliable protection systems.

## Proposed Solution

The system detects motion using PIR sensors. When animal movement is detected, it activates deterrent outputs such as sound and light to scare animals away.

## Components Used

* ESP Microcontroller
* PIR Motion Sensor
* Solar Power Supply
* Buzzer / Speaker
* LED Light Module

## Working Principle

1. PIR sensor monitors movement in the field.
2. When motion is detected:

   * Controller activates sound deterrent
   * LED light flashes randomly
3. Randomized output prevents animals from adapting.

## Features

* Motion-based intrusion detection
* Solar-powered operation
* Randomized deterrent output
* Offline functionality

## Applications

* Crop protection
* Wildlife deterrence
* Rural automation systems
